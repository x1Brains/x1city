On-Chain Assets

NFTs With Smart Metadata

Land, property, and items in X1City are NFTs — but not static ones. Smart metadata makes them dynamic. LP positions become transferable NFTs. The more land you hold, the more you can build on it. Properties can run games and protocols.

Land Parcel NFT
City Plot
Own a piece of X1City. More land = more buildable space. Launch protocols on your parcel. Host games. Rent to other builders. Your footprint in the city scales with your holdings.
Property NFT
Building Deed
Own buildings in any district. Lease them. Run protocols inside. Host games. The NFT showroom in Brains District evolves into a full AMM marketplace — the building grows with the city.
Liquidity NFT
LP Position
Via the Brains LP Pairing Protocol — LP positions can be tokenized as NFTs, allowing an entire pool position to be sold or transferred to a new holder in a single transaction. DeFi meets true ownership.
